# Stratygrafia

Ten projekt służy do pomocy konserwatorom w tworzeniu i eksportowaniu tabeli stratygraficznej.
Jej funkcjonalności obejmują:
 - dodawanie nowej warstwy, która ma swoją przypisaną graficzną reprezentację
 - modyfikacje szczegółów warstwy, takich jak jej procentowość, grubość, faza chronologiczna, datowanie

Aplikacja dynamicznie pokazuje wartości, pozwala zmieniać je dynamicznie w tabeli oraz eksportuje gotową tabelę do pliku docx, którą można potem skopiować do swojej dokumentacji.
